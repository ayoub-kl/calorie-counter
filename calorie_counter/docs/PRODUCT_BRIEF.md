# Product Brief — AI Calorie Counter Android App

## Goal

Build an Android app that lets users log meals from photos with AI-assisted calorie and macro estimation.

## Primary user value

- fast meal logging
- lower friction than manual search entry
- editable AI estimates
- clear meal history

## Core user journey

1. Choose meal type
2. Take meal photo
3. Wait for AI analysis
4. Review estimated foods/calories/macros
5. Correct if needed
6. Save meal
7. Review meal history later

## Product principles

- AI should reduce effort, not remove control
- correction must be easier than starting over
- the app must not overstate certainty
- saved history must feel reliable and readable

## Out of scope for earliest MVP

- advanced social/community features
- full medical guidance
- complex diet coaching engine
- mandatory account flows
- deep wearable/health integrations
